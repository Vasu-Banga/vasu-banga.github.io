VA Engine

Minimalist/Integrated Game Engine Developed Fully In House

Scripting

All scripts go in resouces/component_types. Scripts/Components are written in Lua, and are linked together internally using the Lua and Luabridge packages

Configurations

Files can allow you to configure the feel of your game. Files like game.config and rendering.config allow you to control global parameters, and files in resources/scenes allow you to configure how individual scenes look in your game. All file management and reading is done through rapidjson. Values are loaded in and stored in glm vectors

Input

Input's facilitated through the SDL library. Supports Controller, Mouse, and Controller Input. Uses a Built-in Input Class To Allow Full Input Integration.

Physics

Physics Fully Integrated Through Box2D.
Adding in a RigidBody component to an object will allow for collisions, triggers, and physics operations. Fully configurable through scripts or in the scene.
